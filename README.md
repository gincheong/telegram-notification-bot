# telegram-notification-bot
텔레그램 키워드 알람 봇

카카오톡의 키워드 알림 기능을 본따 만들었습니다.
1. 키워드 알림을 활성화할 그룹 채팅에 봇을 초대합니다.
2. <b>그룹 내 채팅으로</b> 봇에게 /start 명령어를 입력해 그룹 알림을 활성화합니다.
3. <b>봇과의 개인 대화로</b> 키워드를 등록합니다.
